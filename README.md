# 📂 LocalDrive

**LocalDrive** è un sistema leggero e performante per la gestione, l'archiviazione e l'indicizzazione di file direttamente su storage locale. 
È progettato per chi cerca il controllo totale sui propri dati, eliminando la dipendenza dai servizi cloud e offrendo un'interfaccia rapida per l'organizzazione dei file e l'automazione delle operazioni di routine.

## ✨ Caratteristiche

* **Zero-Cloud Dependency:** I tuoi dati rimangono sul tuo hardware. Massima privacy, nessuna latenza di rete.
* **Indicizzazione Intelligente:** Scansione rapida delle directory per trovare i tuoi file in millisecondi.

---

## 🚀 Installazione (Obbligatoria)

Per garantire il corretto funzionamento di tutti i componenti e delle dipendenze di sistema, **non è possibile installare il progetto tramite riga di comando o clonando la repository**. 

Segui questi passaggi per l'installazione corretta:

1.  Vai nella sezione **[Releases](https://github.com/andreailproplayer23-creator/LocalDrive/releases)** di questa repository.
2.  Scarica il file **`LocalDrive_Setup_v1.0.exe`** (o il pacchetto relativo al tuo sistema operativo).
3.  Esegui l'installer e segui le istruzioni a schermo.

> ⚠️ **Nota:** L'utilizzo del setup ufficiale è l'unico modo per configurare correttamente i permessi di scrittura e le variabili d'ambiente necessarie per le automazioni.
